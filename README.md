## Getting Started
Requirements:
- Node.js
- Git
- Code Editor
- Internet Connnection


Step by step:

1. Clone the repository

        git clone 


2. Then, install dependencies:

        npm install


3. Generate database server from prisma console:

        npx prisma generate


4. Finally, run the application:
    
        npm run dev
